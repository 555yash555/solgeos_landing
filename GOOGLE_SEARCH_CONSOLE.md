# Google Search Console Verification

To verify your site with Google Search Console:

1. Go to https://search.google.com/search-console
2. Add property: https://solegoes.in
3. Choose verification method (HTML file or meta tag)
4. If using HTML file method, add the verification file to root
5. If using meta tag, add to index.html <head> section

## Verification Meta Tag Template
```html
<meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE" />
```

## After Verification

### Submit Sitemap
1. In Search Console, go to Sitemaps
2. Submit: https://solegoes.in/sitemap.xml

### Request Indexing
1. Use URL Inspection tool
2. Request indexing for:
   - https://solegoes.in/
   - https://solegoes.in/partner.html

### Monitor Performance
- Check Coverage report
- Monitor Search Performance
- Fix any issues in Core Web Vitals
